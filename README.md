# AgriSahayam - Agricultural Support Platform

AgriSahayam (meaning "Agricultural Aid" in Hindi) is a web-based platform designed to empower farmers with essential tools, information, and resources for better agricultural practices and resource management.

## ğŸš€ Getting Started

### Installation

1. Clone the repository:
```bash
git clone https://github.com/chandana949/agrisahayam.git
cd agrisahayam
```

2. Install all dependencies:
```bash
npm install
```

### Development

Run the development server:
```bash
npm run dev
```

The application will be accessible at `http://localhost:5173`.

### Production Build

Create an optimized production build:
```bash
npm run build
```

Preview the production build locally:
```bash
npm run preview
```
